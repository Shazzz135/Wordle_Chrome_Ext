# Installing the Extension #

## 1. Clone the repo
- Clone the repo and make sure it is a physical accessible folder on your computer

---

## 2. Add the Extension

1. Open "Extensions"  beside your browser profile (Puzzle Shape).

2. CLick on "Manage Extensions".

3. Enable "Developer Mode" in the top right corner of the page.

4. CLick on "Load unpacked".

5. Browse and Select the repo folder.

---

## Notes

Since this extension is not published yet, you can only use it this way. Your developer mode will also need to stay enabled which isn't an issue. Anyways enjoy playing faster.
